# Major changes to the IOCCC entry tool set

## Release 0.1

Created this CHANGES.md markdown document.

Added -T flag to mkiocccentry to print the IOCCC entry tool set release tag:

```sh
./mkiocccentry -T
```

- jstrdecode
- jstrencode
- mkiocccentry
- txzchk

- fnamchk.1
